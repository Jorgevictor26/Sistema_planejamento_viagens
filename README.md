# Sistema_planejamento_viagens

Sistema moderno e profissional de planejamento de viagens desenvolvido com Angular, Laravel API e MySQL.

A plataforma permite que utilizadores:

* Planejem viagens
* Pesquisem destinos reais
* Organizem roteiros
* Controlem despesas
* Visualizem clima
* Explorem destinos em mapas interativos

---
